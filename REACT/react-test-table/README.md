# React Test APP [Table, Light/Dark Mode, Dialog...]
Projeto criado utilizando:
- `vite`
- `react`
- `node js`

#### API 
Executar os comandos
```sh
    cd api
    npm i
    npm run start
```

#### FRONT 
Executar os comandos
```sh
    cd front
    npm i
    npm run dev
```

##### Navegação
Endereço: **http://localhost:5173/**

Páginas:
- `/login`
- `/` (home)
- `/user/new` (cadastro)
- `/user/edit/1` (editar usuário cadastrado)
